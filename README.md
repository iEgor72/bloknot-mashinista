# Shift Tracker App

Это Telegram Web App с общей синхронизацией данных.

## Что умеет

- работает в Telegram и в ярлыке на телефоне;
- хранит смены отдельно для каждого пользователя;
- синхронизирует данные через общий сервер;
- показывает московское время и считает норму по месяцам.

## Как запустить локально

1. Установи Node.js.
2. Открой терминал в этой папке.
3. Выполни:

```bash
npm start
```

4. Открой `http://localhost:3000`.

## Как сделать, чтобы работало для всех

Нужно развернуть этот репозиторий как Node-сервис.

Если используешь Render:

1. Создай новый Web Service из этого репозитория.
2. Render прочитает [`render.yaml`](./render.yaml) автоматически.
3. После деплоя получишь публичный URL.
4. Этот URL нужно указать в Telegram-боте как Web App ссылку.

## Как работает синхронизация

- В Telegram приложение берёт `sid` пользователя Telegram.
- В ярлыке на телефоне используется тот же `sid`, если открыть ссылку из кнопки "Добавить на экран".
- Поэтому Telegram и телефон видят одни и те же смены.

## Важно

- Данные хранятся на сервере в папке `data/`.
- В репозиторий эти данные не коммитятся.
- Если сменить URL или очистить данные на хостинге, список смен тоже изменится.
